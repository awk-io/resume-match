# Match · Resume Matcher

A simple offline tool that compares your resume against a job description.

## Features

- Paste or upload `.txt` / `.pdf` files
- Drag and drop support
- Shows match score (0-100%)
- Lists matched and missing skills
- Suggests specific keywords to add
- Dark / Light mode
- Keyboard shortcut: `Ctrl+Enter` to analyze
- 100% offline – no data uploaded

## How to Use

1. Open `index.html` in any browser
2. Paste job description (left) and resume (right)
3. Click **Analyze** or press `Ctrl+Enter`
4. Review results

## Files

- `index.html` – Structure
- `style.css` – Styles
- `script.js` – Logic

## Technologies

- Vanilla HTML, CSS, JavaScript
- pdf.js for PDF extraction
- Font Awesome for icons
